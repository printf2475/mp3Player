---

👨‍💻 **투입 인원** : 1명

📒 **주요 업무** 

- Content Provider를 사용하여 노래 정보 가져오기
- Recycler View를 사용하여 노래 정보를 보여주고 Glide를 사용하여 이미지 표현
- DB연동하여 플레이 리스트 기능 추가
- 백 그라운드 사용을 위한 Service, Noitfication 사용
